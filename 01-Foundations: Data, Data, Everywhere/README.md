# 01 – Foundations of Data Analytics

## Overview

This module introduces the fundamental concepts of data analytics and the role of a data analyst.

It builds the foundation for understanding how data is used to solve real-world business problems.

---

## Key Concepts Learned

### 1. What is Data Analytics?

Data analytics is the process of collecting, organizing, and analyzing data to discover useful insights that support decision-making.

### 2. Role of a Data Analyst

A data analyst:
- Collects data
- Cleans and prepares data
- Analyzes patterns and trends
- Creates visualizations
- Communicates findings to stakeholders

### 3. Data-Driven Decision Making

Organizations use data to:
- Improve performance
- Reduce risk
- Understand customer behavior
- Increase revenue

---

## Types of Data

### Qualitative (Categorical) Data
Descriptive information that represents characteristics.

Examples:
- Country
- Gender
- Product category

### Quantitative (Numerical) Data
Measurable values.

Examples:
- Sales revenue
- Age
- Number of customers

---

## The Data Analysis Process

The 6-step process introduced in this course:

1. Ask – Define the problem
2. Prepare – Collect data
3. Process – Clean data
4. Analyze – Identify patterns
5. Share – Present insights
6. Act – Make decisions

This structured approach ensures clarity and consistency in analysis.
