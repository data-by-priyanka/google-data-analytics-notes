## Spreadsheet Basics

A spreadsheet is used to organize and analyze data in rows and columns.

- Each rectangular box is called a **cell**.
- Each cell should contain one data value.
- Columns are labeled with letters (A, B, C...).
- Rows are labeled with numbers (1, 2, 3...).
- Each cell has a unique address made from the column letter and row number (for example, A1).

### Important Terms

**Attribute**  
An attribute is a column header. It describes the type of data in that column.

**Observation**  
An observation is a row in a dataset. It contains all the related data for one record.

Spreadsheets help you:
- Organize data
- Sort and filter information
- Perform calculations
- Create charts and graphs

A **formula** is a custom calculation created by the user.  
A **function** is a built-in calculation provided by the spreadsheet.

## Structured Query Language (SQL)

SQL is used to communicate with databases.

A **database** is an organized collection of data.  
A **query** is a request for data from a database.

### Basic SQL Structure

```sql
SELECT column_name
FROM table_name
WHERE condition;
```

- `SELECT` chooses the columns.
- `FROM` specifies the table.
- `WHERE` filters the data.

Important concepts:

- `SELECT *` returns all columns.
- `LIKE` searches for patterns.
- `%` is a wildcard.
- `<>` means not equal.
- `AS` creates an alias.
- `--` is used for comments.

SQL is powerful for working with large datasets.

## Data Visualization

Data visualization means presenting data using charts or graphs.

It helps:
- Show patterns
- Explain trends
- Support decisions

### Common Chart Types

- Line chart → Shows trends over time
- Bar chart → Compares categories
- Pie chart → Shows parts of a whole

### Steps to Create a Visualization

1. Explore the data
2. Decide what message you want to show
3. Create and refine the chart

Keep charts simple, clear, and focused.

---

## Visualization Tools

**Spreadsheets**  
Good for simple charts like bar charts and pie charts.

**Tableau**  
Used to create interactive dashboards.

**R (RStudio)**  
Used for statistical analysis and advanced visualizations.
